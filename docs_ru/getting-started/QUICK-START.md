# Быстрый старт: запуск OmniRoute за 3 минуты

> **TL;DR**: Установить → Подключить бесплатного провайдера → Направить IDE на OmniRoute. Готово.

---

## Шаг 1: Установите OmniRoute

Выберите удобный способ:

### Вариант A: npm (рекомендуется)

```bash
npm install -g omniroute
```

### Вариант B: Docker

```bash
docker run -d --name omniroute -p 20128:20128 diegosouzapw/omniroute:latest
```

### Вариант C: из исходников

```bash
git clone https://github.com/diegosouzapw/OmniRoute.git
cd OmniRoute
npm install
npm run dev
```

---

## Шаг 2: Запустите OmniRoute

```bash
omniroute
```

OmniRoute запускается на `http://localhost:20128`. Панель управления откроется автоматически.

---

## Шаг 3: Подключите бесплатного провайдера

Вы можете пользоваться OmniRoute **совершенно бесплатно**, подключив бесплатного провайдера.

### Вариант A: Kiro (бесплатный Claude — без банковской карты)

1. Откройте панель управления на `http://localhost:20128`
2. Перейдите в **Providers** → **Add Provider**
3. Выберите **Kiro AI**
4. Нажмите **Connect** (API-ключ не нужен!)
5. Готово! Теперь у вас есть бесплатный доступ к моделям Claude.

### Вариант B: OpenCode Free (без авторизации)

1. Откройте панель управления на `http://localhost:20128`
2. Перейдите в **Providers** → **Add Provider**
3. Выберите **OpenCode Free**
4. Нажмите **Connect** (API-ключ не нужен!)
5. Готово! Теперь у вас есть бесплатный доступ к нескольким моделям.

### Вариант C: Pollinations (ключ не нужен)

1. Откройте панель управления на `http://localhost:20128`
2. Перейдите в **Providers** → **Add Provider**
3. Выберите **Pollinations**
4. Нажмите **Connect** (API-ключ не нужен!)
5. Готово! Теперь у вас есть бесплатный доступ к GPT-5, Claude, Gemini и другим.

---

## Шаг 4: Направьте вашу IDE на OmniRoute

В вашей IDE или CLI-инструменте задайте:

```
Base URL: http://localhost:20128/v1
API Key:  [скопируйте из панели управления → Endpoints]
Model:    auto
```

Вот и всё! Теперь ваша IDE использует OmniRoute с автоматическим выбором провайдера.

---

## Шаг 5: Проверьте, что всё работает

```bash
curl http://localhost:20128/v1/models -H "Authorization: Bearer YOUR_KEY"
```

Вы должны увидеть список подключённых моделей.

## Частые вопросы

### «Нужен ли мне API-ключ?»

**Нет!** Вы можете использовать бесплатных провайдеров (Kiro, OpenCode Free, Pollinations) вообще без API-ключа. Просто подключите их в панели управления.

### «Что такое `auto`?»

`auto` указывает OmniRoute автоматически выбирать лучшего провайдера для каждого запроса. Учитываются скорость, стоимость, качество и доступность. Подробности — в [руководстве по Auto-Combo](./AUTO-COMBO-GUIDE.md).

### «Сколько это стоит?»

Сам OmniRoute **бесплатный и открытый**. Вы платите только за провайдеров, которых используете. У многих провайдеров есть бесплатные тарифы — см. [руководство по бесплатным тарифам](./FREE-TIERS-GUIDE.md).

### «Можно ли использовать с Claude Code / Cursor / Copilot?»

**Да!** OmniRoute работает с любым инструментом, поддерживающим формат OpenAI. Просто укажите base URL `http://localhost:20128/v1`. Конкретные инструкции по настройке см. в [руководстве по CLI-инструментам](../reference/CLI-TOOLS.md).

### «Что, если провайдер выйдет из строя?»

OmniRoute автоматически пропускает отказавших провайдеров и пробует следующего. От вас ничего не требуется. Подробности — в [руководстве по Auto-Combo](./AUTO-COMBO-GUIDE.md).
