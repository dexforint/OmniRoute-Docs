# Руководство по бесплатным тарифам: бесплатный ИИ без банковской карты

> **TL;DR**: OmniRoute объединяет бесплатные тарифы более чем 50 провайдеров. Подключите несколько бесплатных провайдеров для безлимитного бесплатного ИИ с автоматическим переключением.

---

## Что такое бесплатные тарифы?

Многие ИИ-провайдеры предлагают **бесплатное использование** — банковская карта не требуется. Это как бесплатные образцы в продуктовом магазине: вы можете попробовать продукт, не платя.

OmniRoute **объединяет** эти бесплатные тарифы в одну точку доступа. Вместо регистрации в 10 разных сервисах вы подключаете их все к OmniRoute и используете `model: "auto"` для автоматического выбора лучшего бесплатного варианта для каждого запроса.

## Как комбинировать бесплатные тарифы

Магия OmniRoute — в **комбинировании бесплатных тарифов**. Вместо того чтобы полагаться на одного провайдера, вы подключаете несколько бесплатных провайдеров и позволяете OmniRoute автоматически выбирать лучшего для каждого запроса.

### Пример: безлимитный бесплатный ИИ

Подключите эти 4 провайдеров для **безлимитного бесплатного ИИ**:

1. **Kiro AI** — 50 кредитов/мес. (модели Claude)
2. **OpenCode Free** — безлимитно (модели GPT)
3. **Pollinations** — ключ не нужен (несколько моделей)
4. **LongCat** — 10M токенов разово (резерв, требуется KYC)

Затем используйте `model: "auto"`, и OmniRoute будет:
- Сначала пробовать Kiro (лучшее качество)
- Если Kiro занят → пробовать OpenCode Free
- Если OpenCode Free медленный → пробовать Pollinations
- Если все отказали → использовать LongCat как резерв

**Результат**: безлимитный бесплатный ИИ с автоматическим переключением!

## Как подключить бесплатных провайдеров

### Шаг 1: откройте панель управления

Перейдите в браузере на `http://localhost:20128`.

### Шаг 2: перейдите в Providers

Нажмите **Providers** в боковой панели.

### Шаг 3: нажмите Add Provider

Нажмите кнопку **+ Add Provider**.

### Шаг 4: выберите бесплатного провайдера

Просмотрите список и выберите одного из этих бесплатных провайдеров:
- **Kiro AI** — бесплатные модели Claude
- **OpenCode Free** — бесплатные модели GPT
- **Pollinations** — бесплатные GPT-5, Claude, Gemini
- **LongCat** — 10M токенов бесплатно (разово, требуется KYC)
- **Cloudflare AI** — 50+ моделей, 10K нейронов/день

### Шаг 5: нажмите Connect

API-ключ не нужен — просто нажмите **Connect**.

### Шаг 6: повторите

Подключите 3–4 бесплатных провайдеров для наилучшего опыта.

## Как OmniRoute улучшает бесплатные тарифы

### 1. Автоматическое переключение

Если один бесплатный провайдер занят или недоступен, OmniRoute автоматически пробует следующего. От вас ничего не требуется.

### 2. Умная маршрутизация

OmniRoute выбирает **лучшего бесплатного провайдера** для каждого запроса на основе:
- Скорости — какой провайдер быстрее прямо сейчас?
- Качества — какой провайдер лучше для этой задачи?
- Ёмкости — у какого провайдера осталась квота?

### 3. Экономия токенов

Функция **сжатия** OmniRoute экономит 15–95% токенов. Это значит, что ваша бесплатная квота длится в **5–20 раз дольше**.

### 4. Поддержка нескольких аккаунтов

Если у вас несколько аккаунтов одного провайдера, OmniRoute рассматривает каждый как отдельного кандидата. Это удваивает или утраивает вашу бесплатную квоту.

## Частые вопросы

### «Это действительно бесплатно?»

**Да!** Это официальные бесплатные тарифы от самих провайдеров. OmniRoute просто упрощает их одновременное использование.

### «Бесплатный тариф закончится?»

У некоторых провайдеров есть лимиты (например, 50 кредитов/мес. у Kiro), но другие безлимитны (например, OpenCode Free и Pollinations). Подключив несколько провайдеров, вы всегда будете иметь резерв.

### «Можно ли использовать бесплатных провайдеров в продакшене?»

**Да!** Многие бесплатные провайдеры готовы к продакшену. Однако для критичных приложений стоит добавить платного провайдера в качестве резерва.

### «В чём подвох?»

Подвоха нет! Провайдеры предлагают бесплатные тарифы для привлечения пользователей. OmniRoute просто упрощает их одновременное использование.

### «Как получить больше бесплатной квоты?»

1. Подключите больше бесплатных провайдеров
2. Используйте сжатие для экономии токенов (экономия 15–95%)
3. Используйте `auto/cheap`, чтобы приоритизировать бесплатных/дешёвых провайдеров
4. Создайте несколько аккаунтов одного провайдера

### «У бесплатных провайдеров качество хуже?»

**Не обязательно!** Многие бесплатные провайдеры предлагают те же модели, что и платные. Например, Kiro даёт доступ к Claude Sonnet 4.5 — той же модели, что и по платной подписке Anthropic.
