# Руководство по провайдерам: подключение ИИ-моделей к OmniRoute

> **TL;DR**: Провайдер — это подключение к ИИ-сервису (например, OpenAI, Anthropic, Google). Для работы с OmniRoute нужен хотя бы один провайдер.

---

## Что такое провайдер?

Провайдер похож на **оператора связи**. Как для звонков нужен оператор, так для использования ИИ-моделей нужен ИИ-провайдер. OmniRoute — это как телефон, который работает со **всеми операторами**: вы можете автоматически переключаться между ними.

### Типы провайдеров

| Тип            | Что это                        | Примеры                           | Стоимость                  |
| -------------- | ------------------------------ | --------------------------------- | -------------------------- |
| **Бесплатный** | Оплата не требуется            | Kiro, OpenCode Free, Pollinations | $0                         |
| **API-ключ**   | Нужен API-ключ                 | OpenAI, Anthropic, Google         | Оплата по использованию    |
| **OAuth**      | Вход через ваш аккаунт         | Claude Code, GitHub Copilot       | Подписка                   |
| **Веб-cookie** | Использует сессию вашего браузера | ChatGPT Web, Gemini Web        | $0 (использует ваш аккаунт) |

## Быстрый старт: подключите первого провайдера

### Вариант A: бесплатный провайдер (без банковской карты)

1. Откройте панель управления на `http://localhost:20128`
2. Перейдите в **Providers** → **Add Provider**
3. Выберите одного из этих бесплатных провайдеров:
   - **Kiro AI** — бесплатные модели Claude (авторизация не нужна)
   - **OpenCode Free** — бесплатные модели GPT (авторизация не нужна)
   - **Pollinations** — бесплатные GPT-5, Claude, Gemini (ключ не нужен)
   - **LongCat** — 10M токенов бесплатно (разовый грант, требуется аккаунт + KYC)
   - **Cloudflare AI** — 50+ моделей, 10K нейронов в день
4. Нажмите **Connect**
5. Готово! Теперь у вас есть бесплатный доступ к ИИ.

### Вариант B: провайдер с API-ключом (платный)

1. Получите API-ключ на сайте провайдера:
   - **OpenAI**: https://platform.openai.com/api-keys
   - **Anthropic**: https://console.anthropic.com/
   - **Google**: https://aistudio.google.com/apikey
   - **DeepSeek**: https://platform.deepseek.com/
   - **Groq**: https://console.groq.com/
2. Откройте панель управления на `http://localhost:20128`
3. Перейдите в **Providers** → **Add Provider**
4. Выберите вашего провайдера
5. Вставьте API-ключ
6. Нажмите **Connect**
7. Готово! Теперь у вас есть доступ к моделям этого провайдера.

### Вариант C: OAuth-провайдер (подписка)

1. Откройте панель управления на `http://localhost:20128`
2. Перейдите в **Providers** → **Add Provider**
3. Выберите провайдера (например, Claude Code, GitHub Copilot)
4. Нажмите **Connect with OAuth**
5. Войдите в свой аккаунт
6. Готово! Теперь у вас есть доступ к моделям вашей подписки.

## Как подключить провайдера (пошагово)

### Шаг 1: откройте панель управления

Перейдите в браузере на `http://localhost:20128`.

### Шаг 2: перейдите в Providers

Нажмите **Providers** в боковой панели.

### Шаг 3: нажмите Add Provider

Нажмите кнопку **+ Add Provider**.

### Шаг 4: выберите провайдера

Просмотрите список или найдите провайдера поиском. Нажмите на него.

### Шаг 5: введите учётные данные

- **Бесплатные провайдеры**: учётные данные не нужны — просто нажмите **Connect**
- **Провайдеры с API-ключом**: вставьте ваш API-ключ
- **OAuth-провайдеры**: нажмите **Connect with OAuth** и войдите

### Шаг 6: проверьте соединение

Нажмите **Test Connection**, чтобы убедиться, что всё работает.

### Шаг 7: готово!

Провайдер подключён. Вы можете использовать его с `model: "auto"` или указать провайдера напрямую.

## Использование нескольких провайдеров

OmniRoute лучше всего работает с **несколькими провайдерами**. Это даёт:

- **Автоматическое переключение** — если один провайдер даёт сбой, OmniRoute пробует следующего
- **Оптимизацию стоимости** — OmniRoute выбирает самого дешёвого провайдера для каждого запроса
- **Оптимизацию скорости** — OmniRoute выбирает самого быстрого провайдера для каждого запроса
- **Оптимизацию качества** — OmniRoute выбирает лучшего провайдера для каждой задачи

### Рекомендуемая конфигурация

Подключите хотя бы **3 провайдеров** для наилучшего опыта:

1. **Одного бесплатного** (Kiro, OpenCode Free или Pollinations) — всегда доступен
2. **Одного быстрого** (Groq, Cerebras) — для быстрых ответов
3. **Одного качественного** (OpenAI, Anthropic, Google) — для сложных задач

Затем используйте `model: "auto"`, и OmniRoute будет автоматически выбирать лучшего для каждого запроса.

## Частые вопросы

### «Нужно ли платить за OmniRoute?»

**Нет!** OmniRoute бесплатный и открытый. Вы можете использовать бесплатных провайдеров (Kiro, OpenCode Free, Pollinations), не платя ничего. Вы платите, только если сами выберете платных провайдеров.

### «С какого провайдера начать?»

Начните с **Kiro AI** — он бесплатный, не требует API-ключа и даёт доступ к моделям Claude. Затем добавляйте других провайдеров по мере необходимости.

### «Можно ли использовать несколько провайдеров одновременно?»

**Да!** В этом и весь смысл OmniRoute. Подключите несколько провайдеров и используйте `model: "auto"`, чтобы OmniRoute выбирал лучшего для каждого запроса.

### «Что, если провайдер выйдет из строя?»

OmniRoute автоматически пропускает отказавших провайдеров и пробует следующего. От вас ничего не требуется.

### «Как отключить провайдера?»

Перейдите в Providers → нажмите на провайдера → нажмите **Disconnect**.

### «Можно ли использовать мои существующие API-ключи?»

**Да!** Если у вас уже есть API-ключи OpenAI, Anthropic, Google и т.д., вы можете использовать их в OmniRoute. Просто вставьте их при подключении провайдера.
