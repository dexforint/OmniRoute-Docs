# Уровни OmniRoute — руководство пользователя

OmniRoute организует 207+ поддерживаемых провайдеров в 3 экономических уровня. Каждый
запрос проходит через них по порядку, пока один не ответит успешно — вы
получаете самый дешёвый из возможных ответов, не написав ни строчки кода переключения.

## Уровень 1 — Подписка

**Провайдеры, за которых вы уже платите.** OmniRoute использует каждую каплю квоты, пока
она не истечёт.

| Провайдер                           | Почему уровень 1                                |
| ----------------------------------- | ----------------------------------------------- |
| Claude Code OAuth                   | Anthropic Pro/Team — фикс. плата, часто простаивает |
| OpenAI Codex (подписка ChatGPT)     | Plus/Team включает квоту Codex                  |
| GitHub Copilot                      | На место — квота сбрасывается ежемесячно        |
| Cursor IDE                          | Квота плана Pro                                 |
| Antigravity / Windsurf              | Встроенные квоты                                |

**Стратегия**: направлять сюда в первую очередь каждый запрос, соответствующий
сильным сторонам модели. Трекер квот следит за приближением сброса; стратегии комбо
`reset-aware` и `subscription` расставляют приоритеты соответственно.

## Уровень 2 — Дешёвый

**Провайдеры с оплатой за токен дешевле $1/1M токенов.** Зарезервированы для больших объёмов
или после исчерпания квот уровня 1.

| Провайдер                     | Цена (ввод/вывод)    | Сильные стороны      |
| ---------------------------- | -------------------- | -------------------- |
| DeepSeek V4 Pro              | $0.27 / $1.10 за 1M  | Код, рассуждения     |
| GLM-4.5                      | $0.60 / $2.20 за 1M  | Длинный контекст     |
| MiniMax M1                   | $0.20 / $1.10 за 1M  | Скорость             |
| Qwen Coder                   | $0.30 / $1.20 за 1M  | Код                  |
| OpenRouter (оптимизация цены) | варьируется         | 100+ моделей, динамика |

**Стратегия**: комбо `cost-optimized` выбирает модель с самой низкой ценой за токен,
которая проходит фильтр возможностей задачи (vision, JSON-режим, инструменты, макс. контекст).

## Уровень 3 — Бесплатный

**Провайдеры с нулевой стоимостью** — бесплатные тарифы, программы кредитов, дневные OAuth-квоты.

| Провайдер        | Бесплатная квота / кредиты              |
| ---------------- | --------------------------------------- |
| Kiro AI          | Бесплатный тариф Claude (щедрый fair-use) |
| OpenCode Free    | Без авторизации, щедрые лимиты частоты  |
| Qoder            | Бесплатный OAuth                        |
| Google Vertex AI | $300 кредитов новому аккаунту           |
| Amazon Q         | Бесплатный тариф для пользователей AWS  |
| Pollinations     | Открытый публичный API                  |
| Cloudflare AI    | Бесплатный тариф Workers AI             |

**Стратегия**: комбо `auto` с ограничением бюджета направляет сюда, когда уровни 1+2 отказали
или когда задано `useFreeOnly=true`. У бесплатных провайдеров часто слабее
лимиты частоты — circuit breaker восстанавливает их по отсрочке.

## Настройка уровней

Панель управления → **Tiers** → назначьте ваших провайдеров. Значения по умолчанию (из `tierDefaults.json`)
разумны; меняйте, когда у вас есть конкретные подписки для приоритизации или провайдеры для исключения.

9-факторный скоринг Auto-Combo также учитывает уровень. См.
[`docs/routing/AUTO-COMBO.md`](../routing/AUTO-COMBO.md).

## Телеметрия

Панель управления → **Usage** показывает токены, потраченные по уровням за день. Используйте это, чтобы:

- Убедиться, что уровень 1 используется полностью (иначе вы теряете ценность подписки)
- Определить, какие модели уровня 2 выбираются чаще всего (консолидируйте до 1–2)
- Убедиться, что уровень 3 экономит деньги на тестовых/исследовательских нагрузках

## Типовые паттерны

### Полностью бесплатная нагрузка

```json
{
  "strategy": "auto",
  "config": { "auto": { "weights": { "costInv": 0.5, "tierPriority": 0.3 } } }
}
```

Сильно смещает к уровню 3; уровень 2 используется, только если уровень 3 недоступен.

### Сначала подписка с дешёвым резервом

```json
{
  "strategy": "priority",
  "targets": [
    { "provider": "claude-code-oauth", "weight": 1 },
    { "provider": "deepseek", "weight": 1 },
    { "provider": "kiro", "weight": 1 }
  ]
}
```

Явный упорядоченный список, повторяющий уровень 1 → уровень 2 → уровень 3.
